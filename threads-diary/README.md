# Threads 合作日誌 - 自動化系統

## 每日流程
1. **21:00** - YuQi 自動生成當日草稿（3個版本）
2. **21:15** - Jacky 選擇/修改版本
3. **21:30** - 發佈到 Threads

## 文件結構
- `DAY-XXX.md` - 每日文案
- `templates/` - 文案模板
- `published/` - 已發佈記錄
