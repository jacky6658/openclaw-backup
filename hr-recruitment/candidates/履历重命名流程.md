## 🔄 未来履历处理流程

### 方法 1：手动指定（适合少量履历）

**步骤**：
1. **上传 PDF** → Telegram 传送履历档案
2. **告诉 YuQi 候选人姓名** → "这是 [候选人姓名]"
3. **YuQi 自动重命名并归档** → 根据分类放到对应文件夹

**示例对话**：
```
你：[上传 PDF]
你：这是 Soun Mara
YuQi：✅ 已重命名为 01-Soun-Mara.pdf 并归档到 P0-优先联系/
```

---

### 方法 2：批量处理（适合大量履历）

**步骤**：
1. **上传所有 PDF** → 一次传送多个履历
2. **提供候选人清单** → 按上传顺序列出姓名
3. **YuQi 批量重命名归档**

**示例清单格式**：
```
file_638 = Soun Mara (P0)
file_639 = Ingrid Ren (P0)
file_640 = Malita Hout (P0)
...
```

或简单版本：
```
1. Soun Mara
2. Ingrid Ren
3. Malita Hout
```

---

### 方法 3：LinkedIn 导出清单（最自动化）

如果 LinkedIn 有导出候选人清单功能（CSV）：
1. **LinkedIn 搜寻** → 找到候选人
2. **导出清单** → 下载 CSV 档案
3. **上传 CSV + PDF** → YuQi 自动对应
4. **自动归档** → 根据 CSV 顺序重命名

## 📋 命名规范

### 档名格式
```
[编号]-[姓-名].pdf

示例：
✅ 01-Soun-Mara.pdf
✅ 04-Chong-Chhe-Chhit.pdf
✅ 其他-Sreyleak-Keo.pdf

❌ SounMara.pdf (缺少连字号)
❌ 01_Soun_Mara.pdf (用底线)
❌ Soun Mara.pdf (有空格)
```

### 编号规则
- **P0 优先联系**：01-03
- **P1 待确认**：04-06
- **P2 备选**：07-11
- **其他-不适合**：前缀"其他-"或"待确认-"
