## 💡 洞察与建议

### 市场洞察
1. **柬埔寨财会人才结构**：
   - NGO/金融业占大宗（~40%）
   - 制造业财会人才相对稀缺
   - 会中文 + 制造业经验 = 极度稀缺

2. **中文能力稀缺性**：
   - LinkedIn搜寻32位，只有8-11位"可能"会中文
   - 端华学校/中国公司背景是关键指标

3. **薪资市场**：
   - NGO平均薪资较低（USD 1,200-1,800）
   - 外商制造业较高（USD 2,000-3,000）
   - 跨国公司Director级（USD 4,000-6,000）

### 建议客户考虑的替代方案

#### 方案1：放宽"中文"要求
- **现况**：会中文候选人仅8-11位
- **替代**：接受"愿意学中文"的候选人
- **配套**：公司提供中文培训、初期配台湾财务副手

#### 方案2：提高薪资包
- **现况**：USD 2,000-2,500（市场中位数）
- **提高至**：USD 3,000-3,500（吸引跨国公司人才）
- **效果**：Ingrid Ren等高阶人才会考虑

#### 方案3：扩大搜寻范围
- **柬埔寨以外**：
  - 越南（华裔社群大）
  - 泰国（曼谷制造业发达）
  - 中国大陆（愿意外派东南亚者）
- **配套**：提供relocation support

#### 方案4：降低经验要求
- **现况**：要求10-15年
- **调整至**：5-8年 + 高潜力
- **例如**：Chong Chhe Chhit（悉尼大学硕士，4年经验）
