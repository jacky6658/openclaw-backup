# 匿名履歷模板 Demo（CV Anonymization Template Demo）

> 用途：履歷進件後，若顧問要「推薦給客戶」前，請 Bot 依此模板產出「匿名版履歷」。
> 原則：去識別個資、保留評估必要資訊；必要時可回填「可聯絡窗口＝Step1ne 顧問」。

---

## 1) 匿名化規則（必刪 / 必改）

### A. 直接移除（不要留下任何痕跡）
- 姓名（中/英）、綽號、英文名
- 手機、電話、Email
- 居住地址（最多保留「縣市/區」層級，例如：台北市／新北市）
- 身分證字號／護照號／生日（生日可改為「年齡區間」）
- 個人照片、簽名檔
- 任何可追溯的連結：個人 FB/IG、Line ID、私人 GitHub（若會暴露姓名）、個人網站（若含姓名）
- 學校信箱、學號、導師姓名、指導教授（若會定位到個人）

### B. 轉換後保留（以「可評估」為主）
- 年齡 → 例如：25–29 / 30–34
- 地點 → 例如：台北市 / 可接受遠端 / 可接受搬遷（選填）
- 學歷 → 保留「學位 + 科系 + 等級」；學校名稱視需求：
  - 預設：隱去校名（只留：國立/私立 + 校級或領域）
  - 若客戶強需求：可保留校名但移除系上導師等細節
- 公司名稱 → 預設匿名化：
  - 例：A 上市電商 / B 外商半導體 / C 新創 SaaS
- 專案/產品名稱 → 如會指向特定公司/客戶，改為類型描述

---

## 2) 匿名履歷輸出格式（建議固定版型）

### 匿名候選人摘要（1頁式）
- **候選人代號**：CAND-YYYYMMDD-001（由 Bot 產生）
- **年資**：X 年（或區間）
- **職能**：例：後端 / 全端 / Data / PM / HR
- **產業經驗**：例：電商、金融、SaaS
- **核心技能**：3–8 個 bullet
- **亮點成就**：2–4 個 bullet（量化優先）
- **期望條件**：薪資區間、地點、到職日（可選）
- **Step1ne 聯絡窗口**：👔 負責顧問：<姓名/代號>（必填）

### 工作經歷（匿名）
每段固定欄位：
- **公司（匿名）/ 產業**：A 外商半導體
- **職稱**：Senior Backend Engineer
- **期間**：2022/07–2025/01
- **工作內容**：3–6 bullets
- **成就**：2–4 bullets（量化）
- **技術/工具**：例：Java, Spring, Kafka, AWS

### 專案節選（可選）
- 專案類型 + 目標 + 貢獻 + 指標

### 學歷 / 證照
- 學位、科系、畢業年份（年份可改區間）

## 4) Bot 使用方式（顧問指令草案）

- 推薦前：
  - `@HRyuqi_bot 匿名履歷：<履歷檔或候選人ID>｜👔負責顧問：<顧問>`
- Bot 回覆：
  - 先回「匿名版預覽（文字）」→ 顧問回 `確認` → 再產出「匿名PDF」或「匿名Doc」
