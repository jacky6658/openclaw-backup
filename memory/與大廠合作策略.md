# 與大廠合作策略 | 不是競爭，是互補

## 大廠現狀分析

### LinkedIn Recruiter
- **優勢**：用戶量大（8.9 億），數據完整
- **劣勢**：爬蟲找人很慢，UI 複雜，價格貴（$50K+/年）
- **痛點**：企業要手動一個一個看，浪費時間

### Indeed
- **優勢**：招聘板做得好，流量大
- **劣勢**：缺乏「主動尋訪」（Passive Recruiting），AI 匹配一般
- **痛點**：很多公司用 Indeed 發職位，但找不到合適的被動候選人

### Workday
- **優勢**：企業 HR 系統整合度高
- **劣勢**：招聘模組很基礎，AI 功能弱
- **痛點**：大企業用 Workday，但往往還要另外裝招聘工具

## 你的 3 種合作模式

### 模式 1：成為 LinkedIn 的「助手工具」（插件 / 整合）

**怎樣做**：
- 開發一個 LinkedIn Recruiter 的「Chrome 外掛」或 API 整合
- 你抓 GitHub 數據，幫用戶快速篩選 LinkedIn 找到的候選人
- 用戶還是在 LinkedIn 上工作，但你加速他們的篩選過程

**例子**：
```
用戶在 LinkedIn 搜尋「Python Engineer」
→ LinkedIn 返回 500 個候選人（太多了）
→ 你的工具自動跳到 GitHub 驗證技能
→ 幫用戶從 500 人縮小到 50 人（有真正的 Python 經驗的）
```

**可行性**：⭐⭐⭐⭐ （很高）

**原因**：
- LinkedIn 允許第三方開發者（有 API）
- 你不是替代 LinkedIn，而是增強它
- LinkedIn 可能甚至願意在應用市場推廣你

**難度**：需要申請 LinkedIn API 訪問權限（商業用途可能要等批准）

**收入模式**：
- 直接賣給用戶 ($3-5K/月)
- 或者 LinkedIn 合作（分潤或內購）

**時間**：2-3 個月開發 + 申請批准

---

### 模式 2：成為 Workday / SAP 的「招聘增強套件」（合作夥伴）

**怎樣做**：
- Workday 有「生態系統合作夥伴計畫」
- 你開發一個 Workday 的「人才尋訪」應用
- 直接整合到客戶的 Workday 系統裡

**例子**：
```
大企業用 Workday 做 HR
→ 他們需要招聘「AI 工程師」
→ Workday 本身沒有被動搜尋功能
→ 他們找到你的應用：「AI 人才搜尋」
→ 一鍵整合，自動從 GitHub 找人 + 導入 Workday
```

**可行性**：⭐⭐⭐ （中等）

**原因**：
- Workday 有官方合作夥伴市場
- 但要通過他們的審核（需要 demo + 用戶反饋）
- 需要技術支援和長期承諾

**難度**：申請流程、集成開發、通過認證

**收入模式**：
- Workday 應用市場分潤（你賺 70-80%）
- 直接賣給客戶（Workday 客戶最願意付費）

**時間**：3-6 個月開發 + 審核

---

### 模式 3：成為企業的「招聘智囊」（諮詢 + SaaS 混合）

**怎樣做**：
- 不競爭工具，競爭「整體解決方案」
- 和大廠企業級客戶（Fortune 500）合作
- 提供「使用我們工具 + 我們的顧問服務」的打包方案

**例子**：
```
大企業：「我們用 LinkedIn Recruiter，但找到的人品質不夠好」
你說：「我們幫你用 AI 驗證 LinkedIn 上的人、爬 GitHub、連接內部 HRIS」
企業願意花 $20-50K/月 因為省了獵頭費（$200K+）
```

**可行性**：⭐⭐⭐⭐⭐ （最高，但最難賣）

**原因**：
- 企業級客戶有預算
- 你不是替代 LinkedIn，而是補強
- 可以包含顧問服務（高利潤）

**難度**：
- 需要銷售團隊 + 顧問
- 決策長（企業採購流程長）
- 但一旦簽了，粘性強、續約率高

**收入模式**：
- SaaS 訂閱：$20-50K/月
- 實施費用：$50-100K（一次性）
- 顧問服務：$300-500/小時

**時間**：3-6 個月準備 + 12 個月銷售週期

## 第四個選項：被大廠收購

**實話：**
- 如果你做得好，LinkedIn / Workday / HireEZ 會主動找你談收購
- 不要一開始就想著被收購，但這是可能的出路
- 被收購通常發生在 $2-5M ARR（年經常性收入）時

**例子**：
- LinkedIn 收購了 CareerBuilder 的一些功能
- Indeed 收購了 Jobspotter（小應用）
- HireEZ 本身是被 Talentdesk 這樣的平台公司投資的

## 具體行動清單

### 申請 LinkedIn 合作

**要求**：
- [ ] LinkedIn API 商業訪問申請
  - 網址：https://business.linkedin.com/talent-solutions/recruitment-tools
  - 需要：公司信息、應用描述、用戶數量
  - 審核時間：1-3 個月

- [ ] 開發 LinkedIn API 整合
  - 用 LinkedIn 的「搜尋 API」
  - 用「職位卡片 API」展示職位
  - 用「郵件 API」發邀請

- [ ] 建立「LinkedIn 應用市場」頁面
  - 演示視頻
  - 用戶推薦
  - 定價方案

### 申請 Workday 夥伴計畫

**要求**：
- [ ] 加入「Workday 應用開發者計畫」
  - 網址：https://www.workday.com/en-us/customer/success/partner-programs.html
  - 需要：簽 NDA、技術評估、安全審查

- [ ] 在 Workday 系統上開發整合
  - 用 Workday API (xRP)
  - 建立 SFDC（Workday 應用市場）應用
  - 通過安全認證

- [ ] 獲得「Workday Certified」狀態

### 準備大客戶銷售

**要求**：

  - ROI 計算器

- [ ] 聘請「銷售代表」或「銷售顧問」
  - 最好找有 HR SaaS 銷售經驗的人
