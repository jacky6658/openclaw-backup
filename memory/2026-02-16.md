#### 修復 3：AI 配對系統（70分鐘）
**問題 1：技能匹配 0 分**
- 候選人：["AI Engineer"]
- JD：["Python", "AI", "Machine Learning", "TensorFlow", "PyTorch", "深度學習"]
- 匹配結果：0/6（0%）

**根本原因**：`normalize_skill()` 把 "AI" 轉成 "artificial intelligence"
- JD: "AI" → "artificial intelligence"
- 候選人: "AI Engineer" → "ai engineer"（小寫）
- 匹配失敗！

**修復 1.1：加入模糊匹配**
```python
def fuzzy_match(jd_skill, candidate_skills):
    jd_lower = jd_skill.lower()
    for c_skill in candidate_skills:
        c_lower = c_skill.lower()
        if jd_lower in c_lower or c_lower in jd_lower:
            return True
    return False
```

**修復 1.2：移除衝突映射**
- 刪除 `'ai': 'artificial intelligence'`
- 保留其他同義詞（'js': 'javascript', 'py': 'python' 等）

**修復 1.3：清理 Python cache**
```bash
find /Users/user/clawd/hr-tools -name "*.pyc" -delete
find /Users/user/clawd/hr-tools -name "__pycache__" -type d -exec rm -rf {} +
```

**成果**：配對分數從 0 → 22.5 分
