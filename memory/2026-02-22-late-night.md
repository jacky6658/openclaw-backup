## 🐛 遇到的挑戰

### 1. Python 套件安裝限制（已解決）
- **問題**: macOS PEP 668 限制 `pip install`
- **解決**: 使用 `--break-system-packages` 或虛擬環境
- **狀態**: 待明天安裝測試

### 2. OpenClaw Claude API 整合（待解決）
- **問題**: `openclaw ask` 命令不存在，`claude` CLI 回傳錯誤
- **嘗試**: 多種整合方式（subprocess、環境變數、API SDK）
- **現狀**: JD 解析器程式碼完成，API 呼叫待優化
- **優先級**: 低（可被需求問卷系統替代）

### 3. Stability Predictor 權重轉換（已解決）
- **問題**: 初版將穩定性 80/100 轉換為 0.2/30（錯誤）
- **修正**: `(score / 100) * 30` → 正確轉換為 24/30
- **驗證**: 測試候選人評分從 56.9 → 80.7 ✅
