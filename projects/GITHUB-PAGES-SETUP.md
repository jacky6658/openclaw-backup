# 如何启用 GitHub Pages

## 快速步骤

1. **打开 GitHub 仓库设置**
   ```
   https://github.com/jacky6658/ai-hr-transformation/settings/pages
   ```

2. **配置 GitHub Pages**
   - Source: 选择 `Deploy from a branch`
   - Branch: 选择 `main`
   - Folder: 选择 `/ (root)`
   - 点击 `Save`

3. **等待 1-2 分钟**
   - GitHub 会自动部署
   - 完成后访问：https://jacky6658.github.io/ai-hr-transformation/

## 验证是否成功

访问以下链接（部署完成后）：
- 主页：https://jacky6658.github.io/ai-hr-transformation/
- 痛点报告：https://jacky6658.github.io/ai-hr-transformation/recruitment-pain-points-analysis.html
- 市场分析：https://jacky6658.github.io/ai-hr-transformation/hr-recruitment-market-analysis-report.html

## 如果还是 404

检查仓库是否为 Public：
1. 进入 https://github.com/jacky6658/ai-hr-transformation/settings
2. 往下滚动到 "Danger Zone"
3. 确认仓库是 `Public`（如果是 Private，GitHub Pages 需要付费）
