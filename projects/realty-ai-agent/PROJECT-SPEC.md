## 🎯 專案目標

建立一套像 Clawbot 一樣聰明、能「自己動起來」的自主開發系統。它不需人類反覆下指令，就能像頂尖房仲一樣，自主在網路上尋找案源、判斷屋主真實性、並在最佳時機發起開發行動。

### 核心需求：自主性 (Autonomy) 是唯一指標

**自主觀察 (Autonomous Scouring)**：
- 系統需像獵犬一樣 24 小時監控全台售屋平台與社群（591、FB、PTT）
- 自動辨識新出現的機會，而非被動等待輸入

### 關鍵功能

1. **24小時自動監控平台**
   - 591 租售屋
   - Facebook 房地產社團
   - PTT 房屋版
   - 自動偵測新物件上架

2. **AI 智能判斷**
   - 屋主 vs 仲介（LLM 分析貼文內容）
   - 物件真實性評估
   - 開發價值評分

3. **自主決策與行動**
   - 基於規則自動發起聯繫
   - 生成客製化開發訊息
   - 自動記錄與追蹤

4. **服務地區**
   - 主要：南投縣 竹山鎮
   - 可擴展至其他地區

## 🛡️ 反爬蟲機制分析

### 常見的反爬蟲手段

1. **IP 限流**（最常見）
   - 同一 IP 短時間內請求過多 → 封鎖

2. **User-Agent 檢測**
   - 辨識爬蟲工具（如 Python requests）

3. **JavaScript 渲染檢測**
   - 需要執行 JS 才能看到內容

4. **CAPTCHA 驗證碼**
   - 人機驗證

5. **行為分析**
   - 偵測「不像人類」的行為（速度太快、軌跡異常）
