   訓練 → 部署 → 監控全流程
   大企業搶人的高端技能

💡 給轉職者的建議：

【從 0 開始】
• 先學 Prompt Engineering（免費資源多）
• 再上手開源模型（LLaMA, Mistral）
• 做 1-2 個 AI 項目上傳 GitHub

【有 Python 基礎】
• 直奔 LLM 微調 + RAG（市場最缺）
• 學 Hugging Face 生態
• 研究企業級應用場景

【有深度學習背景】
• 轉向 MLOps（天花板高）

❌ 只學理論，不做項目

• Prompt Engineer：80-120k（新手）

• AI Agent 架構師：150-240k（資深）
• MLOps 工程師：130-200k（高度緊缺）

- 是什麼：訓練模型適應特定任務
- 企業用途：客服、內容生成、知識問答
- 薪資：120-180k

- 學習資源：OpenAI Cookbook, LangChain

- 是什麼：自主決策 + 循環執行的 AI 系統
- 企業用途：自動化工作流、RPA 升級版
- 薪資：150-240k
- 學習資源：LangChain, AutoGPT, CrewAI

Slide 5: Vector DB
- 是什麼：針對向量資料優化的數據庫
- 企業用途：RAG 系統的記憶庫
- 薪資：110-160k
- 學習資源：Pinecone, Chroma, Weaviate

Slide 6: MLOps
- 是什麼：生產級 AI 系統的工程化
- 企業用途：訓練、部署、監控、迭代
- 薪資：130-200k （高端技能）
- 學習資源：DVC, MLflow, Kubeflow

Slide 7: 轉職路線圖
新手 → Prompt Eng → LLM 微調 → AI Agent / MLOps
