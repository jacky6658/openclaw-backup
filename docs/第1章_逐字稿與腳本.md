# 影片 1-1｜環境準備：安裝 Node.js、確認系統需求（5 分鐘）

## 逐字稿
大家好，我是 Jacky。
這一段我們要完成 **Node.js 安裝**，這是 OpenClaw 能在電腦上運行的基礎環境。
不需要程式背景，你只要照著我做就可以。

第一步，打開瀏覽器，進入 nodejs.org。
你會看到 LTS 與 Current 兩個版本，請選 **LTS（穩定版）**。
下載完成後，雙擊安裝檔，一直按「繼續」「安裝」即可。

安裝完成後，打開「終端機」。
輸入：node --version
如果看到版本號，代表安裝成功。
再輸入：npm --version
同樣看到版本號就沒問題。

到這裡，Node.js 已經安裝完成。

# 影片 1-2｜OpenClaw 本機安裝：一鍵安裝指令與驗證（10 分鐘）

## 逐字稿
Node.js 裝好後，我們開始安裝 OpenClaw。
請打開終端機，輸入：

npm install -g openclaw

你會看到文字跑過去，這是正常的。
等到出現「added」或「packages」字樣，就代表安裝完成。

接著輸入：
openclaw --version
如果看到版本號，就代表 OpenClaw 已經裝好。

再來做初始化設定：
openclaw init

系統會問幾個問題：
- 工作目錄（直接按 Enter 用預設）
- 模型供應商（先選一個，之後可再改）
- API Key（可以先跳過）

完成後，你會看到初始化成功的訊息。

如果出現權限錯誤，請在指令前加 sudo：
sudo npm install -g openclaw

這一步完成後，下一段我們要建立 Telegram Bot。

# 影片 1-3｜Telegram 連接設定：建立 Bot、取得 Token、測試對話（10 分鐘）

## 逐字稿
OpenClaw 裝好後，我們需要一個「對話介面」，最簡單的是 Telegram。

打開 Telegram，搜尋 @BotFather。
輸入 /start，再輸入 /newbot。

第一步輸入 Bot 名稱，例如：Jacky AI Assistant。
第二步輸入 Bot 帳號，必須以 bot 結尾，例如：jacky_ai_2026_bot。

成功後你會看到一串 Token，請務必保密。

回到終端機，輸入：
openclaw config set telegram.botToken "你的Token"

接著輸入：
openclaw gateway start

看到「Gateway running」就代表啟動成功。

回到 Telegram，對你的 Bot 輸入 /start。
如果收到回應，就表示綁定完成。

下一段會教你設定 AI 模型與 API Key。

# 影片 1-4｜AI 模型選擇與設定：GPT / Claude / Gemini、API Key（5 分鐘）

## 逐字稿
這一段要完成 AI 模型設定。
你可以選 GPT、Claude 或 Gemini。
簡單說：
- GPT：穩定、用途廣
- Claude：寫作與推理好
- Gemini：速度快、成本低

你只要準備好 API Key。
回到終端機，依需求輸入：
openclaw config set openai.apiKey "你的Key"
或
openclaw config set anthropic.apiKey "你的Key"

設定完後，重新啟動 OpenClaw，測試對話。
