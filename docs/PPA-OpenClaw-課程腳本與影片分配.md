## 影片 2｜Node.js 安裝（5 分鐘）
**標題**：第 1 章-1：安裝 Node.js（OpenClaw 的運行環境）

**開場**：Node.js 是讓 OpenClaw 能跑起來的基礎環境。\
**步驟**：
1. 前往 nodejs.org（LTS 版本）下載
2. 安裝 .pkg（一路下一步）
3. 終端機輸入 `node --version`、`npm --version` 驗證

**結尾**：Node.js 完成，下一步安裝 OpenClaw。

---

## 影片 3｜OpenClaw 本機安裝（10 分鐘）
**標題**：第 1 章-2：一鍵安裝 OpenClaw + 驗證成功

**步驟**：
1. `npm install -g openclaw`
2. `openclaw --version` 驗證
3. `openclaw init` 初始化（workspace / model provider / API key 可先略）

**常見錯誤**：
- Permission denied → `sudo npm install -g openclaw`
- command not found → 重開終端機

**結尾**：下一步建立 Telegram Bot。

---

## 影片 4｜Telegram Bot 設定（10 分鐘）
**標題**：第 1 章-3：建立 Telegram Bot + 第一次對話測試

**步驟**：
1. Telegram 搜尋 @BotFather → /newbot
2. 建立 bot name + username（以 bot 結尾）
3. 取得 Token（務必保密）
4. `openclaw config set telegram.botToken "..."`
5. `openclaw gateway start`
6. Telegram 對 Bot /start 測試對話

**結尾**：AI 已上線，下一步設定模型 API Key。

---

## 影片 5｜模型與 API Key 設定（5 分鐘）
**標題**：第 1 章-4：GPT / Claude / Gemini 模型設定

**流程**：
1. 各模型 API Key 取得位置（快速口頭說明）
2. `openclaw config set openai.apiKey "..."`（示範一種）
3. `openclaw config set anthropic.apiKey "..."`（示範一種）
4. `openclaw config set defaultModel "gpt-4o-mini"`（或實際可用模型）

**結尾**：完成模型設定，進入備份章節。
