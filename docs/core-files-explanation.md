# 核心檔案說明

> 本文件說明了 Clawdbot 工作區中核心檔案的用途。

## 檔案說明

*   **AGENTS.md：** 描述了 Agent 的工作區，包含 Agent 的行為準則、記憶管理、安全措施、群組聊天規則、工具使用方式和心跳機制等。
*   **BOOTSTRAP.md：** Agent 首次啟動時的引導文件，指示 Agent 如何了解自己、與使用者互動，以及設定基本資訊。
*   **HEARTBEAT.md：** 描述 Agent 如何利用心跳來進行定期檢查，以及如何主動完成工作。
*   **IDENTITY.md：** 記錄 Agent 的身份資訊，例如：名稱、本質、風格和表情符號。
*   **SOUL.md：** 定義 Agent 的核心價值觀、行為準則和個性，是 Agent 的靈魂所在。
*   **TASKS.md：** 記錄 Agent 目前的任務清單和執行狀態，方便 Agent 追蹤和管理任務進度。
*   **TOOLS.md：** 記錄 Agent 可用的工具及其相關資訊，例如：攝影機名稱、SSH 主機、TTS 偏好設定等。
*   **USER.md：** 記錄使用者的相關資訊，例如：姓名、時區、聯絡方式和偏好設定等。
