關鍵控制點（v2.0 更新）

  • 步驟1A（新增）：公司背景研究（產品、服務、文化、技術環境）
  • 步驟2（擴充）：建立完整人才畫像
    • ✅ 產品契合度（優先產業經驗）
    • ✅ 文化契合度（價值觀、工作風格）
    • ✅ 為什麼候選人會想來？（吸引力分析）
    • ✅ 什麼樣的人會成功？（成功特質）
  • 步驟2 用戶確認（新增）：人才畫像必須經用戶確認才能搜尋
  • 步驟5：強制執行資料清理（移除 newline + 替換逗號）
  • 步驟6-7：測試 1 筆 + 等用戶確認才批量
  • 步驟9：最終驗證，有錯誤立刻回滾

v2.0 新增檢查點

檢查點 檢查內容 失敗處理
公司分析完成 是否了解產品、文化、技術環境？ 執行 web_search 補充資訊
用戶確認人才畫像 畫像是否符合公司需求？ 調整畫像重新提交
文化契合度 候選人是否符合企業文化？ 排除不符合候選人

常見失敗點與防護（v2.0 更新）

失敗點 原因 v2.0 防護措施
找到不符合公司文化的人 未研究公司背景 步驟1A 強制公司研究
找到國外開發者 未確認 location 步驟2 強制確認「台灣本地」
資料被拆成多行 逗號被當成換行 步驟5 強制 replace(', ', ' | ')
欄位錯位 newline 字元殘留 步驟5 驗證無 newline
批量匯入失敗 未測試就批量 步驟6-7 測試1筆+用戶確認

完整人才畫像範本（v2.0）

公司背景分析

  • 公司名稱、規模、產業
  • 主要產品與服務
  • 企業文化與價值觀
  • 技術環境與團隊

為什麼候選人會想來？

  • 產業地位與品牌
  • 產品挑戰與成長機會
  • 企業文化吸引力
  • 職涯發展空間

什麼樣的人會成功？

  • 產品契合度（產業經驗）
  • 文化契合度（價值觀、工作風格）
  • 技能要求（必備 + 加分）
  • 成功特質（主動、溝通、學習力）

目標候選人畫像

  • 基本條件（地點、年資、語言）
  • 經驗背景（職位、產業）
  • 技能要求（必備 + 加分）
  • 排除條件（太資深、國外、不符合文化）
