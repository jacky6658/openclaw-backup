# 緊急切換模型指南（不在電腦旁時）

## 🚨 遇到模型 Cooldown 時

當你看到這個錯誤：
```
All models failed (rate_limit / cooldown)
```

---

## 📱 方法 1：Telegram 直接切換（最快）

直接在 Telegram 發送以下任一指令：

### 切換到 Gemini Pro（推薦，100% 可用）
```
/model gemini-2.5-pro
```

### 切換到 Gemini Flash（更快更省）
```
/model gemini-2.5-flash
```

### 切換到 Antigravity Claude（用 Claude Max 配額）
```
/model google-antigravity/claude-opus-4-5-thinking
```

### 切回預設模型（Claude Opus - Antigravity）
```
/model default
```

### 或直接指定切回 Claude
```
/model google-antigravity/claude-opus-4-5-thinking
```

### 切回 Claude Sonnet (API)
```
/model sonnet
```

### 切回 Claude Opus (API)
```
/model opus
```

---

| 指令 | 模型 | 特點 | 配額 |
|------|------|------|------|
| `/model gemini-2.5-pro` | Gemini Pro | 平衡 | ✅ 100% |

## ⚡ 緊急處理流程（SOP）

**看到 cooldown 錯誤時**：

1. 發送：`/model gemini-2.5-pro`
2. 等待 10 秒（模型切換需要時間）
3. 發送任何訊息測試（例如：`hi`）
4. ✅ 應該就能正常回覆了

## 📞 緊急聯絡

如果切換後還是不行：
1. 發送：`/restart`（重啟 agent）
2. 或等待 5-10 分鐘讓 cooldown 自動解除
3. 或直接發「yuqi」叫醒我檢查

---

## 🎯 推薦設定

**日常使用建議**：
- 主力：`google-antigravity/claude-opus-4-5-thinking`（用 Claude Max）
- 備用：`gemini-2.5-pro`（無配額限制）
- 省錢：`gemini-2.5-flash`（快速任務）

**cooldown 時自動順序**：
1. Gemini Pro（100% 可用）
2. Gemini Flash（更快）
3. 等待 Claude 解除 cooldown
