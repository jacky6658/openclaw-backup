# OpenClaw 課程拍攝口白（逐段提詞）

> 目的：每支影片除了開場外的「講解口白」
> 格式：章節 / 小節 / 影片編號 + 口白要點

---

## 影片 1｜第 0 章 單元 1–4（合併）
**主題：開場與準備**
- 說清楚 OpenClaw 是什麼、為什麼需要（本機 AI、Telegram、記憶、技能）
- 說清楚課程路徑（安裝 → 備份 → 排錯 → 擴充）
- 說明準備清單（Mac、網路、Telegram、API Key、硬碟空間）
- 強調「不需程式背景、照做即可」

---

## 影片 2｜第 1 章 單元 1
**主題：Node.js 安裝**
- Node.js 是 OpenClaw 的運行環境
- 下載 LTS 版本、安裝完成
- 終端機用 `node --version`、`npm --version` 驗證

---

## 影片 3｜第 1 章 單元 2
**主題：OpenClaw 本機安裝**
- `npm install -g openclaw` 安裝
- `openclaw --version` 驗證
- `openclaw init` 初始化（workspace / provider）
- 常見錯誤：權限、command not found

---

## 影片 4｜第 1 章 單元 3
**主題：Telegram 綁定**
- @BotFather 建立 bot（/newbot）
- 取得 Token、提醒不要外洩
- `openclaw config set telegram.botToken "..."`
- `openclaw gateway start` 啟動
- Telegram 對話測試 /start

---

## 影片 5｜第 1 章 單元 4
**主題：模型與 API Key 設定**
- GPT / Claude / Gemini 的差異（品質 vs 成本）
- 教學 API Key 放哪裡
- 示範 config set（擇一或兩個）
- 測試一輪對話

---

## 影片 6｜第 2 章 單元 2
**主題：完整備份流程**
- 備份目的：記憶與設定不丟失
- 示範備份指令
- 打開備份資料夾講結構（memory / config / media）
- 提醒備份不要上傳公開平台

---

## 影片 7｜第 2 章 單元 3
**主題：換機還原**
- 新電腦先安裝 OpenClaw
- 將備份放回正確位置
- 啟動驗證記憶是否回來
- 舊機關閉 gateway、清理

---

## 影片 8｜第 3 章 單元 2
**主題：Port 佔用排查**
- Port 是「門牌號碼」的概念
- 18789 被占用會導致啟動失敗
- 示範查詢與釋放 port
- 重啟 gateway 驗證

---

## 影片 9｜第 3 章 單元 3
**主題：記憶修復**
- MEMORY.md / memory/ 是 AI 長期記憶
- 示範修復步驟
- 提醒先備份再修
- 測試是否恢復記憶

---

## 影片 10｜第 4 章 單元 1
**主題：Skills 管理**
- 什麼是 Skill（擴充功能）
- ClawHub 搜尋與安裝
- 安裝後簡單測試

---

## 影片 11｜第 4 章 單元 2
**主題：Google Workspace 整合**
- 安裝 / 設定 MCP
- OAuth 流程簡述
- 測試 Gmail / Calendar / Drive

---

## 影片 12｜教材下載（GitHub）
**主題：模板庫下載與使用**
- GitHub 下載 ZIP
- 解壓後資料夾結構
- 怎麼指示 AI 讀檔學習

---

## 影片 13｜結語
**主題：總結與下一步**
- 回顧學到的 4 大能力（安裝 / 備份 / 排錯 / 擴充）
- 提醒定期備份與更新
- 鼓勵加入社群、分享成果
