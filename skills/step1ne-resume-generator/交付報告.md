```bash
cd /Users/user/clawd/skills

```

**Step 2：閱讀文件**
```bash
cd step1ne-resume-generator
cat README.md      # 快速開始
cat SKILL.md       # 完整說明
cat MANIFEST.md    # 學習路徑
```

**Step 3：實作練習**
```bash
# 使用範本產出第一份履歷
cp templates/resume-template.html /tmp/test-resume.html
# （手動編輯 /tmp/test-resume.html）
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome \
  --headless --disable-gpu \
  --print-to-pdf=/tmp/test.pdf \
  --print-to-pdf-no-header \
  /tmp/test-resume.html 2>/dev/null
open /tmp/test.pdf
```
