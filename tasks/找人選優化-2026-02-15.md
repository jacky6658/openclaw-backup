### 1. AI 配對演算法升級 ✅

**目標**：從單一關鍵字匹配 → 多維度智慧評分

**評分模型（0-100 分）**
```python
總分 = 技能匹配度 (40%) + 經驗年資 (30%) + 產業相關性 (20%) + 其他加分項 (10%)

# 技能匹配度 (40%)
- 必備技能完全符合：+40
- 必備技能部分符合：+20-35（按比例）
- 加分技能符合：+5-10

# 經驗年資 (30%)
- 符合 JD 要求年資：+30
- 略低於要求（-1年）：+20
- 遠低於要求（-2年以上）：+10
- 超過要求（senior overqualified）：+25

# 產業相關性 (20%)
- 同產業同職位：+20
- 同產業不同職位：+15
- 相關產業同職位：+10
- 無相關：+0

# 其他加分項 (10%)
- GitHub 活躍（commits/stars）：+5
- 知名公司背景：+3
- 技術社群活躍：+2
```

**紅旗檢測（扣分項）**
- 頻繁跳槽（1年內換3家）：-15
- 技能完全不符：-20
- 地點不符且無遠端經驗：-10

**信心分級**
- P0（高度符合）：總分 ≥ 80
- P1（可能符合）：總分 60-79
- P2（待確認）：總分 40-59
- 不推薦：總分 < 40

### 3. 去重邏輯強化 ✅

**問題**：
- 同一人可能在 LinkedIn、GitHub、履歷池都出現
- 沒有追蹤「已推薦」歷史，可能重複推薦

**解決方案**：

**候選人指紋（Fingerprint）**
```python
def generate_fingerprint(candidate):
    # 姓名標準化（去空格、轉小寫、繁簡轉換）
    name_normalized = normalize_name(candidate['name'])
    
    # 公司標準化（去 "股份有限公司"、"有限公司"）
    company_normalized = normalize_company(candidate['company'])
    
    # 組合指紋
    fingerprint = f"{name_normalized}_{company_normalized}"
    return fingerprint
```

**跨平台去重**
1. 讀取所有候選人（LinkedIn + GitHub + 履歷池）
2. 計算每個人的 fingerprint
3. 合併重複項目（保留資訊最完整的）
4. 補充跨平台資訊（LinkedIn profile + GitHub repos）

## 整合流程（最終版）

**每週一 10:00 自動執行**
```bash
#!/bin/bash
# auto-sourcing-v2.sh

# 1. 讀取所有「招募中」職缺
jds=$(gog sheets read --sheet-id 1QPaeOm-slNVFCeM8Q3gg3DawKjzp2tYwyfquvdHlZFE | grep "招募中")

# 2. 對每個職缺執行多管道搜尋
for jd in $jds; do
  # 2.1 LinkedIn
  bash google-linkedin-search.sh "$jd_title" "$jd_skills"
  
  # 2.2 GitHub
  python3 github-talent-search.py --keywords "$jd_skills" --location Taiwan
  
  # 2.3 CakeResume
  python3 scraper-cakeresume.py --query "$jd_title" --skills "$jd_skills"
  
  # 2.4 Yourator
  python3 scraper-yourator.py --query "$jd_title"
  
  # 2.5 PTT Soft_Job
  python3 scraper-ptt-softjob.py --keywords "$jd_skills"
  
  # 2.6 履歷池內部搜尋
  bash search-resume-pool.sh "$jd_skills"
done

# 3. 合併所有結果
python3 merge-candidates.py --output /tmp/all-candidates.json

# 4. 去重
python3 dedup-engine.py --input /tmp/all-candidates.json --output /tmp/deduped.json

# 5. AI 配對評分
python3 ai-matcher-v2.py --jd "$jd_id" --candidates /tmp/deduped.json --output /tmp/scored.json

# 6. 排序 + 篩選 Top 10
cat /tmp/scored.json | jq 'sort_by(-.score) | .[0:10]' > /tmp/top10.json

# 7. 匯入履歷池
python3 import-to-resume-pool.py --input /tmp/top10.json

# 8. 發送 Telegram 通知（Topic 304）
python3 notify-telegram.py --topic 304 --jd "$jd_id" --candidates /tmp/top10.json
```
