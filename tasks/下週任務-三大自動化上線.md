### 3. 自動找人選開發（新功能）
**新腳本 `auto-sourcing.sh`**:
```bash
#!/bin/bash
# 使用方式: ./auto-sourcing.sh <JD_ID>

JD_ID=$1

# 1. 讀取 JD（從 Google Sheets）
jd_data=$(gog sheets get --sheet-id "1QPaeOm..." --range "A$JD_ID:H$JD_ID")

# 2. AI 提取關鍵字
keywords=$(echo "$jd_data" | openclaw ai extract-keywords)

# 3. 平行搜尋
# 3.1 GitHub（技術職）
github_results=$(bash ~/clawd/hr-tools/github-talent-search.sh "$keywords")

# 3.2 履歷池配對
resume_results=$(python ~/clawd/hr-tools/batch_match.py --jd-id $JD_ID)

# 3.3 Google 搜尋（目標公司）
google_results=$(python ~/clawd/hr-tools/google-search-candidates.py "$keywords")

# 4. AI 統一配對 + 排序
all_candidates="$github_results\n$resume_results\n$google_results"
ranked=$(echo "$all_candidates" | openclaw ai rank-candidates --jd "$jd_data")

# 5. 輸出 Top 20
echo "$ranked" | head -20 > /tmp/candidates-$JD_ID.json

# 6. 發送 Telegram 通知
openclaw message send --message "🔍 自動找人選報告

職位: $(echo "$jd_data" | jq -r '.title')

✅ 找到候選人: $(echo "$all_candidates" | wc -l) 個
🏆 Top 5:
$(echo "$ranked" | head -5 | jq -r '.name + " (" + .score + "分)"')

📊 完整清單: file:///tmp/candidates-$JD_ID.json"
```

**新腳本 `google-search-candidates.py`**:
```python
#!/usr/bin/env python3
import sys
from googlesearch import search

keywords = sys.argv[1]
query = f"{keywords} site:linkedin.com/in OR site:104.com.tw/profile"

results = []
for url in search(query, num_results=50):
    # 爬取 LinkedIn/104 個人頁面
    profile = scrape_profile(url)
    results.append(profile)

print(json.dumps(results))
```
