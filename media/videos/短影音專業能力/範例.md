# 短影音專業能力 - 範例（給新 Bot 學習）

## 最佳學習順序（新 Bot）
1. 先讀「規則.md」
2. 再讀「SKILL.md」
3. 讀本檔範例，直接模仿輸出格式

---

## 範例 1｜流量型（問句 Hook）
**主題**：新手用 ChatGPT 做簡報
**目標**：流量
**受眾**：20–35 歲上班族
**素材**：螢幕錄影 + 人說話

**標題**：
「為什麼你用 ChatGPT 做簡報，主管還是不買單？」

**Hook（0–5s）**：
「你以為 ChatGPT 做完就能交？錯，關鍵在這 3 個步驟。」

**Value（≤3 點）**：
1) 「先給框架，再給內容」
2) 「每頁只留一個重點」
3) 「用數字收尾更有說服力」

**CTA**：
「想拿模板？留言『我要』我傳你。」

**拍攝/剪輯**：
- 開頭 CU + 大字卡「3 個步驟」
- 每點 2–3 秒切畫面

---

## 範例 2｜轉單型（反差 Hook）
**主題**：AI 行銷課程
**目標**：轉單
**受眾**：創業者
**素材**：講師本人 + 成果截圖

**標題**：
「只花 7 天，我的轉單率從 2% 變 10%」

**Hook（0–5s）**：
「7 天，把轉單率翻 5 倍，我只做這一件事。」

**Value（≤3 點）**：
1) 「先改文案，不改廣告」
2) 「用 A/B 測試找最高轉換」
3) 「用案例做信任」

**CTA**：
「想學這套？點下方連結。」

---

## 範例 3｜信任型（Problem → Solution → Proof）
**主題**：短影音沒人看
**目標**：信任/教育
**受眾**：自媒體新手
**素材**：人說話 + before/after 數據

**Hook**：
「你不是不會剪，是開場 3 秒就輸了。」

**Value**：
1) 「先丟結果，不要鋪陳」
2) 「字幕只留關鍵詞」
3) 「每 2 秒換一個畫面」

**Proof**：
「我照做後，完播率從 12% → 41%」

**CTA**：
「想要檢查清單？留言『清單』。」

## Q&A Flow（讓 Bot 逐步提問）
Q1 結構（選 1）：A 三段式 / B 問題解決 / C Before-After / D 教學 / E 敘事 / F 爆點連發
Q2 主題或產品名稱（例：美白咀嚼錠）
Q3 主要目標（吸流量 / 教育/信任 / 轉單 / 品牌）
Q4 目標受眾（年齡/性別/特質/痛點）
Q5 可用素材（Before/After / 專家/素人講話 / 日常切片 / 截圖 / 無）
Q6 開場鉤子類型（問句 / 反差 / 同理 / 數字）+ 關鍵詞
Q7 CTA（關注/收藏 / 留言/私訊 / 購買連結）
