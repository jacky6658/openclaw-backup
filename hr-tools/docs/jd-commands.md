# JD 管理指令（群組可用）

## 📋 可用指令

### 查看職缺
- `列出職缺` / `JD列表` / `職缺列表`
  - 顯示所有職缺
  
- `搜尋職缺 [關鍵字]`
  - 例如：`搜尋職缺 AI`
  - 搜尋包含關鍵字的職缺

- `JD統計` / `職缺統計`
  - 顯示職缺統計報表

### 新增職缺
- `新增JD [職位] [客戶] [部門] [人數] [薪資] [技能] [經驗] [學歷] [地點]`
  - 例如：`新增JD DevOps工程師 AIJob內部 技術部 1 90k-130k Docker,K8s,AWS 3年以上 大學以上 台北`

### 更新職缺
- `更新JD [行號] [狀態]`
  - 例如：`更新JD 2 已成交`
  - 狀態選項：開放中、暫停、已成交、取消

---

## 🎯 使用範例

**查看所有職缺**
```
列出職缺
```

**搜尋特定職缺**
```
搜尋職缺 工程師
```

**新增職缺**
```
新增JD 後端工程師 AIJob內部 技術部 2 80k-120k Python,FastAPI,PostgreSQL 3年以上 大學以上 台北
```

**更新狀態**
```
更新JD 3 已成交
```
