# LinkedIn 個人檔案優化指南

## 🎯 優化目標
讓候選人看到你的檔案時：
1. 立刻知道你是做什麼的
2. 感覺專業、可信賴
3. 想要聯繫你

---

## 📸 1. 個人照片（Profile Photo）

### ❌ 避免
- 生活照、旅遊照
- 模糊、昏暗的照片
- 戴墨鏡、背對鏡頭

### ✅ 建議
- **專業形象照**（正裝 or Smart Casual）
- **微笑、自信**的表情
- **純色背景**（白色、淺灰、淺藍）
- **高解析度**（至少 400x400 px）

💡 **快速方案**：
- 用手機拍（自然光、乾淨背景）
- 用 remove.bg 去背
- 用 Canva 加專業背景

---

## 🎨 2. 封面照片（Banner）

### 建議尺寸
- 1584 x 396 px（LinkedIn 官方尺寸）

### 設計元素
```
[左側] 文字標語
「幫助工程師找到夢想工作」
「Tech Talent | Career Growth | Step1ne」

[右側] 視覺元素
- Step1ne Logo
- 職涯階梯圖示
- 數據統計（例：「已協助 200+ 工程師成功轉職」）
```

💡 **Canva 模板**：搜尋「LinkedIn Banner」

## 📖 4. 關於（About）

### 結構

```
[第一段 - Hook]
你是不是也遇過這些問題？
• 投了 50 封履歷，只有 3 個面試機會
• 不知道自己的市場價值是多少
• 想轉職，但不知道從何開始

我是 Jacky，Step1ne 的資深獵頭顧問。
過去 X 年，我協助了 200+ 位工程師找到理想工作。

[第二段 - 專業領域]
我專注在：
✅ 科技業人才招募（.NET / Python / AI / PM）
✅ 薪資談判與職涯規劃
✅ 隱性職缺推薦（90% 職缺不會公開）

[第三段 - 價值主張]
與我合作，你會得到：
🎯 客製化職缺推薦（不是海投）
💰 薪資行情分析與談判建議
🤝 終身職涯顧問（不只是一次性服務）

[第四段 - Call to Action]
💼 正在找新機會？私訊我，讓我們聊聊。
📧 Email: jacky@step1ne.com
🦞 一起找到你的下一個里程碑！

[Hashtags]
#獵頭 #科技業招募 #職涯規劃 #工程師 #Step1ne
```

**字數限制**：2600 字元（中文約 1000 字）

## 🏆 7. 推薦（Recommendations）

### 如何獲得推薦？

**主動請求**：
```
Hi [候選人名字]，

很高興能協助你成功拿到 [公司] 的 offer！

如果你覺得我的服務有幫助，可以在 LinkedIn 上給我一個推薦嗎？
這會幫助更多人找到理想工作。

謝謝！
Jacky
```

**推薦內容範例**（給候選人參考）：
```
Jacky 是我遇過最專業的獵頭顧問。
他不只是推職缺，更會花時間了解我的職涯目標。
在他的協助下，我成功拿到 3 個 offer，薪資成長了 40%。
強烈推薦！
```
