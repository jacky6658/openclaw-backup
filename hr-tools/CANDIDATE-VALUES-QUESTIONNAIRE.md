# 候選人價值觀問卷

**用途**：評估候選人工作價值觀，用於公司文化適配度分析  
**填寫對象**：求職候選人  
**時間**：5-10 分鐘  
**填寫時機**：投遞履歷後、面試前

---

## 📋 10 個核心問題

### 1. 工作價值優先級
**Q1: 在工作中，你最看重的 3 個方面是？**（按優先級排序）

請從以下選項中選擇 3 個，並標示 1-3 的優先順序：

- [ ] 薪資福利（具競爭力的薪資與獎金）
- [ ] 職涯發展（快速晉升或技術成長機會）
- [ ] 工作穩定（公司穩定，不易裁員）
- [ ] 企業文化（價值觀契合，氛圍舒適）
- [ ] Work-Life Balance（彈性工時，生活品質）
- [ ] 工作挑戰（有趣的專案，技術挑戰）
- [ ] 團隊氛圍（與同事相處融洽）
- [ ] 公司願景（認同公司使命與發展方向）

---

### 2. 工作節奏偏好
**Q2: 你偏好的工作節奏是？**

- [ ] 快速變化（喜歡新挑戰，適應力強）
- [ ] 穩定可預測（喜歡按計劃執行，不喜歡頻繁變動）
- [ ] 混合型（可以適應，但有偏好）→ 偏好：___________

---

### 3. 團隊氛圍偏好
**Q3: 你喜歡的團隊氛圍是？**

- [ ] 競爭型（喜歡良性競爭，個人績效導向）
- [ ] 協作型（喜歡團隊合作，共同達成目標）
- [ ] 平衡型（既有個人表現也有團隊合作）

---

### 4. 決策與自主性
**Q4: 在工作中，你偏好？**

- [ ] 高度自主（自己決策，自己負責）
- [ ] 聽從指導（主管給明確指示，按指示執行）
- [ ] 協作決策（與團隊/主管討論後共同決定）

---

### 5. 加班態度
**Q5: 你對加班的態度是？**

- [ ] 完全不接受（堅持準時下班，work-life balance 優先）
- [ ] 偶爾可接受（忙季或專案期間可以，但不希望常態）
- [ ] 可以接受（願意為工作投入額外時間）
- [ ] 主動加班（享受工作，願意長時間投入）

---

### 6. 溝通風格
**Q6: 你的溝通風格是？**

- [ ] 直接坦率（有話直說，快速解決問題）
- [ ] 委婉含蓄（注重禮貌，避免衝突）
- [ ] 視情況調整（對不同人用不同方式）

---

### 7. 職涯目標
**Q7: 你的職涯目標是？**（可複選，最多 2 個）

- [ ] 快速晉升（3-5 年內成為主管/總監）
- [ ] 技術專家（深耕技術領域，成為專家）
- [ ] 創業準備（累積經驗與人脈，未來創業）
- [ ] 平衡生活（穩定工作，享受生活）
- [ ] 多元發展（嘗試不同領域與角色）
- [ ] 財務自由（賺取足夠財富後退休/轉換跑道）

---

### 8. 對失敗的態度
**Q8: 當工作中犯錯時，你的反應是？**

- [ ] 從錯誤中學習（接受失敗，快速調整）
- [ ] 謹慎避免犯錯（事前規劃，降低風險）
- [ ] 壓力很大（擔心犯錯，需要時間調適）

---

### 9. 工作動力來源
**Q9: 什麼最能激發你的工作動力？**（最多選 2 個）

- [ ] 挑戰性任務（困難的問題，技術突破）
- [ ] 明確目標（清楚的 KPI，達成成就感）
- [ ] 團隊認同（同事肯定，團隊氛圍）
- [ ] 薪資獎勵（獎金、加薪、股票）
- [ ] 學習成長（學習新技術，提升能力）
- [ ] 社會影響（工作有意義，幫助他人）

---

### 10. 過去離職原因
**Q10: 你過去離職的主要原因是？**（可複選，最近 2 次）

- [ ] 薪資不滿意
- [ ] 缺乏發展機會
- [ ] 加班太多，work-life balance 差
- [ ] 與主管/同事不合
- [ ] 公司文化不適應
- [ ] 公司不穩定（裁員、營運問題）
- [ ] 工作內容不符合期待
- [ ] 搬家/個人因素
- [ ] 其他：___________

## 🎯 適配度分析範例

當候選人與公司文化比對後：

```
候選人：張三
公司：XXX 科技

文化適配分析：
✅ 高度匹配（90-100%）
  - 工作節奏：候選人「快速變化」vs 公司「快速迭代」
  - 溝通風格：候選人「直接坦率」vs 公司「直接坦率」
  - 失敗態度：候選人「從錯誤中學習」vs 公司「高度容忍」

⚠️ 中度匹配（50-89%）
  - 加班文化：候選人「偶爾可接受」vs 公司「偶爾加班」
  - 團隊氛圍：候選人「協作型」vs 公司「協作導向」

❌ 不匹配（<50%）
  - Work-Life Balance：候選人優先級 3，但公司「工作優先」

總體適配度：75/100（B 等級）
風險提示：候選人重視 work-life balance，但公司可能有加班需求，
         建議面試時明確溝通工作時間與彈性。
```
