# Google Forms 問卷設置指南

**目的**：建立公司文化問卷 + 候選人價值觀問卷  
**平台**：Google Forms  
**預估時間**：15 分鐘

---

## 📋 問卷 1：公司文化問卷

### 基本設定

- **標題**：公司文化評估問卷
- **描述**：
  ```
  感謝您填寫此問卷！

  這份問卷將幫助我們更準確地為您推薦適合的候選人。
  問卷約需 5-10 分鐘完成，所有資料將嚴格保密。

  填寫對象：HR 主管或用人主管
  ```

### 問題清單（10 題）

#### Q1: 公司願景與使命
- **類型**：簡答題
- **問題**：請簡述貴公司的願景或使命？
- **說明**：例如：「成為台灣最創新的 AI 公司」「提供穩定可靠的金融服務」
- **必填**：否

---

#### Q2: 核心價值觀
- **類型**：核取方塊（可複選，最多 3 個）
- **問題**：貴公司最重視的核心價值觀是？（最多選 3 個）
- **選項**：
  - 創新突破（鼓勵嘗試新事物）
  - 穩定可靠（注重流程與品質）
  - 快速執行（效率優先）
  - 團隊協作（重視合作精神）
  - 個人績效（強調個人成果）
  - 客戶至上（以客戶需求為中心）
  - 技術領先（追求技術卓越）
  - 成本控制（注重效率與節約）
- **必填**：是
- **驗證**：最多 3 項

---

#### Q3: 工作節奏
- **類型**：單選題
- **問題**：貴公司的工作節奏是？
- **選項**：
  - 快速迭代（快速變化，頻繁調整）
  - 穩定發展（按計劃推進，變化較少）
  - 混合型（有快有慢，視專案而定）
- **必填**：是

---

#### Q4: 決策風格
- **類型**：單選題
- **問題**：貴公司的決策風格是？
- **選項**：
  - 扁平化（鼓勵所有人提意見，決策透明）
  - 階層化（由主管決策，層級分明）
  - 混合型（視情況而定）
- **必填**：是

---

#### Q5: 團隊氛圍
- **類型**：單選題
- **問題**：貴公司的團隊氛圍偏向？
- **選項**：
  - 競爭導向（鼓勵良性競爭，個人表現）
  - 協作導向（強調團隊合作，共同目標）
  - 平衡型（既有競爭也有協作）
- **必填**：是

---

#### Q6: 加班文化
- **類型**：單選題
- **問題**：貴公司的加班情況是？
- **選項**：
  - 幾乎無加班（正常上下班）
  - 偶爾加班（忙季或專案期間，1-2 天/月）
  - 經常加班（1-2 天/週）
  - 常態加班（幾乎每天）
- **必填**：是
- **標記**：⚠️ 重要評估項目

---

#### Q7: 溝通方式
- **類型**：單選題
- **問題**：貴公司的溝通文化是？
- **選項**：
  - 直接坦率（有話直說，快速解決）
  - 委婉含蓄（注重禮貌，間接表達）
  - 正式規範（會議記錄，正式流程）
  - 非正式彈性（Slack/即時通訊為主）
- **必填**：是

---

#### Q8: 成長路徑
- **類型**：單選題
- **問題**：貴公司的職涯發展特色是？
- **選項**：
  - 快速晉升（表現優異可快速升遷）
  - 穩定發展（按年資與經驗逐步晉升）
  - 專業深耕（鼓勵成為領域專家）
  - 多元發展（可跨部門輪調）
- **必填**：是

---

#### Q9: 失敗容忍度
- **類型**：單選題
- **問題**：貴公司對失敗的態度是？
- **選項**：
  - 高度容忍（鼓勵嘗試，從錯誤中學習）
  - 中度容忍（可以失敗，但需檢討改進）
  - 低度容忍（強調避免犯錯，穩定為主）
- **必填**：是

---

#### Q10: Work-Life Balance
- **類型**：單選題
- **問題**：貴公司對工作與生活平衡的重視程度？
- **選項**：
  - 非常重視（彈性工時、遠端工作、不鼓勵加班）
  - 適度重視（正常上下班，偶爾加班可接受）
  - 工作優先（專案優先，需要時全力投入）
- **必填**：是
- **標記**：⚠️ 重要評估項目

---

### 表單設定

- **回覆接收**：Google Sheets（自動建立試算表）
- **回覆限制**：僅限一次回覆
- **確認訊息**：
  ```
  感謝您的填寫！

  我們將根據您提供的資訊，為您推薦最適合的候選人。
  如有任何問題，歡迎隨時與我們聯繫。
  ```

## 📋 問卷 2：候選人價值觀問卷

### 基本設定

- **標題**：工作價值觀評估問卷
- **描述**：
  ```
  您好！

  這份問卷將幫助我們更準確地為您推薦適合的工作機會。
  問卷約需 5-10 分鐘完成，所有資料將嚴格保密。

  請依照您的真實想法填寫，沒有對錯之分。
  ```

### 問題清單（10 題）

#### Q1: 工作節奏偏好
- **類型**：單選題
- **問題**：您偏好的工作節奏是？
- **選項**：
  - 快速變化（喜歡新挑戰，適應力強）
  - 穩定可預測（喜歡按計劃進行，不喜歡突然改變）
  - 混合型（可快可慢，視情況調整）
- **必填**：是

---

#### Q2: 決策參與度
- **類型**：單選題
- **問題**：您偏好的決策方式是？
- **選項**：
  - 高度自主（希望自己做決定，承擔責任）
  - 協作決策（希望與團隊一起討論決定）
  - 聽從指導（偏好主管給明確指示）
- **必填**：是

---

#### Q3: 團隊合作風格
- **類型**：單選題
- **問題**：您偏好的團隊氛圍是？
- **選項**：
  - 競爭型（喜歡與同事良性競爭，激發潛力）
  - 協作型（喜歡團隊合作，共同達成目標）
  - 平衡型（既享受競爭也重視合作）
- **必填**：是

---

#### Q4: 加班接受度
- **類型**：單選題
- **問題**：您對加班的接受程度是？
- **選項**：
  - 完全不接受（重視準時下班，嚴格遵守工時）
  - 偶爾可接受（忙季或專案期間可配合，但不希望常態化）
  - 可以接受（願意為工作需要加班）
  - 主動加班（享受投入工作的感覺，不介意加班）
- **必填**：是
- **標記**：⚠️ 重要評估項目

---

#### Q5: 溝通風格
- **類型**：單選題
- **問題**：您偏好的溝通方式是？
- **選項**：
  - 直接坦率（喜歡有話直說，快速解決問題）
  - 委婉含蓄（重視禮貌，間接表達意見）
  - 正式流程（偏好會議記錄與正式溝通）
  - 非正式彈性（喜歡用 Slack 或即時通訊）
- **必填**：是

---

#### Q6: 職涯發展期待
- **類型**：單選題
- **問題**：您期待的職涯發展路徑是？
- **選項**：
  - 快速晉升（希望快速升遷，承擔更大責任）
  - 穩定發展（按部就班，穩健成長）
  - 專業深耕（成為領域專家，不一定要管理職）
  - 多元發展（嘗試不同領域，跨部門學習）
- **必填**：是

---

#### Q7: 失敗容忍度
- **類型**：單選題
- **問題**：您對工作失敗的接受程度是？
- **選項**：
  - 高度接受（失敗是學習機會，勇於嘗試）
  - 中度接受（可以接受失敗，但希望檢討改進）
  - 低度接受（希望避免犯錯，偏好穩健策略）
- **必填**：是

---

#### Q8: Work-Life Balance
- **類型**：單選題
- **問題**：您對工作與生活平衡的重視程度？
- **選項**：
  - 生活優先（重視個人時間，工作是謀生手段）
  - 平衡型（工作與生活同等重要）
  - 工作優先（願意為工作全力投入，生活可以妥協）
- **必填**：是
- **標記**：⚠️ 重要評估項目

---

#### Q9: 工作動力來源
- **類型**：單選題
- **問題**：您工作的主要動力是？
- **選項**：
  - 成就感（完成挑戰性任務的滿足感）
  - 穩定收入（薪資與福利保障）
  - 成長機會（學習新技能，拓展視野）
  - 工作生活平衡（有足夠時間陪伴家人朋友）
  - 社會影響力（對社會有貢獻）
- **必填**：是

---

#### Q10: 最重視的工作條件
- **類型**：核取方塊（可複選，最多 3 個）
- **問題**：以下工作條件中，您最重視哪些？（最多選 3 個）
- **選項**：
  - 高薪資
  - 完善福利
  - 成長空間
  - 工作穩定
  - 公司文化
  - 工作彈性（遠端/彈性工時）
  - 團隊氛圍
  - 主管領導風格
- **必填**：是
- **驗證**：最多 3 項

---

### 表單設定

- **回覆接收**：Google Sheets（自動建立試算表）
- **回覆限制**：僅限一次回覆
- **確認訊息**：
  ```
  感謝您的填寫！

  我們將根據您的回覆，為您推薦最適合的工作機會。
  如有任何問題，歡迎隨時與我們聯繫。
  ```

## 📊 後續處理

### 1. 設定回覆試算表

**公司文化問卷**：
- 試算表名稱：`公司文化問卷回覆 - [公司名稱]`
- 欄位對應：
  - A: 時間戳記
  - B: 公司願景
  - C: 核心價值觀
  - D: 工作節奏 → `work_pace`
  - E: 決策風格 → `decision_style`
  - F: 團隊氛圍 → `team_atmosphere`
  - G: 加班文化 → `overtime_culture` ⚠️
  - H: 溝通方式 → `communication_style`
  - I: 成長路徑 → `growth_path`
  - J: 失敗容忍度 → `failure_tolerance`
  - K: Work-Life Balance → `work_life_balance` ⚠️

**候選人價值觀問卷**：
- 試算表名稱：`候選人價值觀問卷回覆`
- 欄位對應：
  - A: 時間戳記
  - B: 工作節奏偏好 → `work_pace_preference`
  - C: 決策參與度 → `decision_preference`
  - D: 團隊合作風格 → `team_preference`
  - E: 加班接受度 → `overtime_acceptance` ⚠️
  - F: 溝通風格 → `communication_preference`
  - G: 職涯發展期待 → `growth_preference`
  - H: 失敗容忍度 → `failure_acceptance`
  - I: Work-Life Balance → `balance_preference` ⚠️
  - J: 工作動力來源 → `motivation`
  - K: 最重視的工作條件 → `work_priorities`

### 2. 整合到系統

**將問卷回覆轉換為 JSON**（使用 `form-to-jd.py` 模式）：

```bash
# 公司文化
python3 culture_matcher.py \
  --company-sheet "公司文化問卷回覆" \
  --output company_culture.json

# 候選人價值觀
python3 culture_matcher.py \
  --candidate-sheet "候選人價值觀問卷回覆" \
  --output candidate_values.json
```

### 3. 文化匹配計算

```bash
python3 stability_predictor_v2.py \
  --candidate candidate_with_career.json \
  --company company_culture.json \
  --mode full
```

## ✅ 建立步驟

### 步驟 1：創建 Google Forms

1. 前往 https://forms.google.com
2. 點擊「新增空白表單」
3. 按照上述問題清單建立問卷
4. 設定回覆接收方式（Google Sheets）

### 步驟 2：測試問卷

1. 預覽表單
2. 測試填寫一次
3. 檢查 Google Sheets 回覆格式

### 步驟 3：分享問卷

**公司文化問卷**：
- 對象：客戶公司 HR
- 方式：Email 寄送連結

**候選人價值觀問卷**：
- 對象：候選人
- 方式：簡訊或 Email

---
