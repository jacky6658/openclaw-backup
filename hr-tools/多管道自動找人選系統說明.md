# 多管道自動找人選系統說明

## ✅ 已完成（2026-02-13）

### 1. GitHub Talent Search（2 小時）✅
**位置**：`/Users/user/clawd/hr-tools/active/tools/github-talent-search.py`

**功能**：
- 搜尋 GitHub 開發者（依地點、語言、repos、followers）
- 提取用戶資料（username, email, bio, location, company, website, twitter）
- 自動判斷技術職位適用的程式語言

**用法**：
```bash
# 基本搜尋
python3 github-talent-search.py taipei python 10

# 參數說明
# $1 = 地點 (taipei, taiwan, etc.)
# $2 = 程式語言 (python, javascript, java, etc.)
# $3 = 最多結果數 (預設 10)
```

**輸出**：
- JSON 格式：`/tmp/github-talent-{location}-{language}-{timestamp}.json`
- 包含：username, url, email, location, company, repos, followers

### 2. CakeResume 搜尋（1 小時）✅

```bash
# $2 = 地點（台灣、台北、新竹等）
# $3 = 最多結果數（預設 15）
```

**輸出**：
- JSON 格式：`/tmp/cakeresume-candidates-{timestamp}.json`
- 匯入格式：`/tmp/cakeresume-formatted-{timestamp}.json`

### 3. 自動分配管道（技術 → GitHub，非技術 → LinkedIn）✅
**位置**：`/Users/user/clawd/hr-tools/active/automation/auto-sourcing-multi-channel.sh`

**功能**：
- 自動讀取「step1ne 職缺管理」中所有「招募中」職缺
- 根據職位類型智能選擇搜尋管道：
  - **技術職** → GitHub + LinkedIn
  - **設計/創意職** → CakeResume + LinkedIn
  - **管理/非技術職** → LinkedIn only
- 多管道結果合併與去重
- 自動匯入履歷池

**判斷邏輯**：

#### 技術職關鍵字 → GitHub + LinkedIn
```
工程師|程式|開發|Developer|Engineer|Software|
Backend|Frontend|Full Stack|DevOps|AI|
Machine Learning|Data|測試|QA|架構師
```

#### 設計/創意職關鍵字 → CakeResume + LinkedIn
```
設計|Designer|UI|UX|創意|Creative|
內容|Content|社群|Community|編輯|Editor
```

#### 管理/非技術職關鍵字 → LinkedIn only
```
經理|主管|VP|總監|Manager|Director|Chief|
顧問|Consultant|分析師|Analyst|產品|Product|
專案|Project|業務|Sales|行銷|Marketing|人資|HR
```

**用法**：
```bash
# 處理所有招募中職缺
bash auto-sourcing-multi-channel.sh

# 從指定行開始
bash auto-sourcing-multi-channel.sh 2

# 優先處理特定職位
bash auto-sourcing-multi-channel.sh 2 "工程師"
```

**流程**：
1. 讀取「step1ne 職缺管理」
2. 篩選「招募中」職缺
3. 根據職位類型選擇管道
4. 執行搜尋（LinkedIn / GitHub / CakeResume）
5. 合併結果並去重
6. 自動匯入履歷池
7. 發送 Telegram 通知（Topic 304）

### 4. 履歷池匯入工具✅
**位置**：`/Users/user/clawd/hr-tools/active/tools/import-multi-channel-results.sh`

**功能**：
- 統一格式化 LinkedIn、GitHub、CakeResume 結果
- 自動去重（比對 LinkedIn URL / Email / 姓名）
- 匯入到履歷池 Google Sheets

**用法**：
```bash

# 指定來源類型
bash import-multi-channel-results.sh /tmp/linkedin-xxx.json "職位名稱" linkedin
bash import-multi-channel-results.sh /tmp/github-xxx.json "職位名稱" github
bash import-multi-channel-results.sh /tmp/cakeresume-xxx.json "職位名稱" cakeresume
```
