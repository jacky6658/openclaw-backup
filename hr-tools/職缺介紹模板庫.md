# 科技獵頭職缺介紹模板庫
Tech Recruiter Job Introduction Templates

本文件提供科技獵頭在 **電話 / LinkedIn / 視訊** 與候選人交流時的 **30 秒職缺介紹模板**。

目標：

- 讓候選人 **快速理解機會**
- 避免冗長公司介紹
- 建立自然對話
- 適用於所有科技職缺

---

# 通用介紹框架

推薦使用以下結構：

```
機會 → 公司 → 工作內容 → 技術 / 能力 → 工作模式 → 回問
```

說明：

|階段|目的|
|---|---|
機會 | 讓候選人立刻知道職位 |
公司 | 提供產業背景 |
工作內容 | 說明角色定位 |
技術 / 能力 | 工程師最關心 |
工作模式 | on-call / remote / 工時 |
回問 | 建立對話 |

# 科技獵頭三個重要技巧

## 1. 一定要提「系統類型」

工程師很在意系統類型，例如：

- 交易系統
- SaaS 平台
- 廣告系統
- AI 平台
- 資料平台

例：

```
金融交易系統
高併發廣告系統
大型 SaaS 平台
```

---

## 2. 技術不要講太多

錯誤：

```
Java Spring Redis Kafka MySQL Docker AWS Terraform...
```

正確：

```
Java / Spring Boot
Kafka
Redis
```

---

## 3. 一定要有回問

因為：

```
回問 = 對話開始
```

範例：

- 這樣的技術環境你會有興趣了解看看嗎？
- 你現在比較想找什麼樣的技術團隊？
- 你現在工作比較偏 infra 還是 platform？
