## ⚠️ 重要注意事項

### LinkedIn 爬蟲
- 需要有效的 LinkedIn Session Cookie
- 建議使用 Brave Search API 作為備援
- 避免短時間大量請求（防止封鎖）

### GitHub 爬蟲
- 使用 GitHub Token 可提升速率限制（5000次/小時）
- 不使用 Token 只有 60次/小時
- Token 設定：`~/.openclaw/openclaw.json` 中的 `GITHUB_TOKEN`

### 591 爬蟲
- 使用 agent-browser 抓取動態網頁
- 需要安裝 Node.js 依賴（JSDOM）
