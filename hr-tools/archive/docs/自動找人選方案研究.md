**研究日期**: 2026-02-12  
**目標**: 讓 YuQi 自動搜尋並推薦候選人

---

## 🎯 核心需求

### 輸入
- JD（職缺描述）
- 關鍵技能/經驗要求
- 地點/薪資範圍

### 輸出
- 候選人清單（姓名、聯絡方式、履歷摘要）
- AI 配對分數
- 推薦理由

### 成功標準
- **準確度**: 推薦候選人 ≥70% 符合 JD
- **效率**: 每個職缺 <30 分鐘找到 10+ 候選人
- **合規**: 不違反平台 ToS、個資法

### Phase 2: 付費方案（1-3個月，視成效決定）

**D. 104 人力銀行 API（台灣市場）**
**成本**: ~30,000 TWD/年  
**功能**: 搜尋求職者、查看履歷、主動邀約

**ROI 評估**:
- 如果每月成功推薦 1 個候選人（收費 20%，年薪 80 萬）
- 年收入 = 16 萬 × 12 個月 = 192 萬
- 年成本 = 3 萬
- **ROI = 6,300%** 🚀

**決策點**: 當履歷池 + GitHub 無法滿足需求時考慮

---

**E. LinkedIn Recruiter Lite**
**成本**: ~$170 USD/月 (~5,000 TWD)  
**功能**: 進階搜尋、InMail 配額（30/月）、候選人追蹤

**ROI 評估**:
- 如果每月成功推薦 1 個候選人（收費 20%，年薪 100 萬）
- 月收入 = 20 萬
- 月成本 = 5,000
- **ROI = 3,900%** 🚀

**決策點**: 當需要挖角海外或高階人才時考慮

---

### Phase 3: 規模化方案（3-6個月）

**F. 自建候選人資料庫（Data Lake）**
```
來源: GitHub + 104 + LinkedIn + 履歷池
結構: PostgreSQL + Vector DB (for AI matching)
自動化: 每日爬蟲 + AI 分析 + 自動配對
```

**優點**:
- ✅ 完全掌控資料
- ✅ AI 配對效果最好
- ✅ 可以做趨勢分析

**缺點**:
- ❌ 開發成本高（1-2 個月）
- ❌ 維護成本（伺服器 + 資料更新）
- ❌ 合規風險（個資保護）

**決策點**: 當業務量 >10 個活躍職缺時考慮

## 🛠️ 立即可用的技術方案

### 方案 1: 多渠道搜尋 + AI 整合

**流程**:
```
1. 輸入 JD → AI 提取關鍵字（技能、經驗、地點）
2. 平行搜尋:
   - GitHub (if 技術職)
   - 履歷池配對
   - Google 搜尋目標公司員工
3. AI 統一打分 + 排序
4. 輸出 Top 10 候選人 + 聯絡方式
```

**實作時間**: 2-3 天  
**成本**: $0  
**效果**: 每個職缺可找到 5-15 個候選人

---

### 方案 2: 目標公司挖角自動化

**流程**:
```
1. 識別競爭對手公司（例如：醫療用品 → 台灣 Top 20）
2. 爬取公司官網「團隊介紹」頁面
3. 爬取 LinkedIn 公司頁面員工列表
4. AI 配對職位 + 經驗
5. 生成聯絡訊息（LinkedIn InMail or Email）
```

**實作時間**: 3-5 天  
**成本**: $0（LinkedIn 免費版 3 InMails/月）  
**效果**: 針對性強，轉換率高

**範例**（你今天做的）:
- 志邦企業需要財會主管 → 搜尋 Cambodia 製造業財會人才
- 美德醫療需要供應鏈 → 搜尋台灣醫療產業供應鏈主管

---

### 方案 3: 社群平台自動監控

**流程**:
```
1. 監控 LinkedIn/Facebook 求職社群
2. 關鍵字過濾（例如：「轉職」「求職」「面試」）
3. AI 配對 JD
4. 自動發送邀約訊息
```

**實作時間**: 1 週  
**成本**: $0  
**效果**: 被動等待，但候選人意願高

## 🤖 AI 自動化程度分級

### Level 1: AI 輔助（目前狀態）
- AI 配對履歷池候選人
- AI 生成聯絡訊息
- **人工**: 搜尋候選人、聯繫候選人

### Level 2: 半自動化（1-2 週可達成）
- AI 自動搜尋 GitHub + 履歷池
- AI 自動配對 + 排序
- AI 生成聯絡訊息
- **人工**: 確認候選人、發送訊息

### Level 3: 全自動化（1-2 個月）
- AI 自動搜尋多渠道
- AI 自動配對 + 排序
- AI 自動發送訊息
- AI 自動追蹤回覆
- **人工**: 只需審核推薦結果
